Introduction

Pronounced C-sharp (not C-hashtag), it is a programming language characterized by being multi-model that features strong typing, functional, declarative, imperative, generic, object-oriented and component-oriented disciplines. It was created in 2000 by Microsoft as part of its .NET initiative. The free and open-source project is called Mono which develops a compiler and runtime for the language. C# is a programming languages which was designed for the Common Language Infrastructure (CLI). C# was designed by Anders Hejlsberg, and Mads Torgersen is the leader of its development team. The most recent version is 8.0, which was released in 2019 alongside Visual Studio 2019 version 16.3
